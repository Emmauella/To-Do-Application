# To-Do Application

A simple To-Do list application built with vanilla JavaScript.

## Features

- Add new tasks
- Remove existing tasks
- Mark tasks as done
- Prevent duplicate tasks
- Persist data in browser LocalStorage

## How to Run

1. Open `index.html` in a web browser
2. Or run a local server: `python3 -m http.server 8000` and visit `http://localhost:8000`

## Files

- `index.html`: The main HTML structure
- `style.css`: Basic styling
- `script.js`: JavaScript logic for the application
